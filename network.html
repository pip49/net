1과목 네트워크 일반 - 고빈출 30제 퀴즈

문제를 읽고 머릿속으로 답을 떠올린 후 '정답 확인' 버튼을 눌러보세요.

📌 학습 가이드: 1과목(네트워크 일반)은 주로 OSI 7계층의 특징, 프로토콜 명칭, 그리고 전송 방식의 기본 개념이 출제됩니다. 단답형 퀴즈를 통해 핵심 키워드를 완벽하게 외워보세요!
Q1. 물리적 매체를 통해 통신 단위인 비트(Bit)를 흐르게 하며, 허브나 리피터가 속한 OSI 7계층은?
물리 계층 (Physical Layer / 1계층)
Q2. 인접한 노드 간 신뢰성 있는 데이터 전송을 보장하며, MAC 주소를 사용하여 스위치나 브리지가 동작하는 OSI 7계층은?
데이터 링크 계층 (Data Link Layer / 2계층)
Q3. 데이터를 목적지까지 가장 안전하고 빠른 경로(Routing)를 선택하여 전송하는 계층으로, 라우터가 작동하는 OSI 7계층은?
네트워크 계층 (Network Layer / 3계층)
Q4. 송수신 프로세스 간 전송 연결을 설정하고, 신뢰성 있는 데이터 전송을 보장하기 위해 흐름 제어 및 오류 제어를 수행하는 OSI 7계층은?
전송 계층 (Transport Layer / 4계층)
Q5. 수신한 전기 신호를 재생, 증폭하여 다음 선로로 전달함으로써 전송 거리를 연장해 주는 1계층 장비는?
리피터 (Repeater)
Q6. 한 포트로 들어온 데이터를 연결된 모든 포트로 그대로 복제해서 뿌려주는 통에 네트워크 효율을 떨어뜨리는 1계층 장비는?
허브 (Hub) 또는 더미 허브
Q7. 허브와 달리 목적지 MAC 주소를 가진 포트로만 데이터를 정확히 정렬해 보내주어 충돌을 방지하는 2계층 장비는?
스위치 (Switch) 또는 스위칭 허브
Q8. 신호가 동시에 전송될 때 발생하는 충돌(Collision)의 영향이 미치는 네트워크 범위를 뜻하는 용어는?
콜리전 도메인 (Collision Domain)
Q9. LAN 영역에서 라우터를 거치지 않고 브로드캐스트 메시지가 도달할 수 있는 전체 영역을 뜻하는 용어는?
브로드캐스트 도메인 (Broadcast Domain)
Q10. 물리적으로 하나인 스위치를 논리적으로 쪼개어 여러 개의 독립된 네트워크 영역으로 분리하는 기술은?
VLAN (Virtual Local Area Network)
Q11. 데이터를 양방향으로 동시에 전송할 수 있는 통신 방식으로, 전화기나 대다수 현대 네트워크 환경에서 쓰이는 전송 방식은?
전이중 통신 (Full-Duplex)
Q12. 양방향 통신은 가능하지만, 무전기(워키토키)처럼 동시에 전송할 수는 없고 번갈아 가며 전송해야 하는 통신 방식은?
반이중 통신 (Half-Duplex)
Q13. 송신 측과 수신 측이 같은 클록(타이머) 신호에 맞추어 블록 단위로 데이터를 전송하는 고속 통신 방식은?
동기식 전송 (Synchronous Transmission)
Q14. 데이터 뒤에 오류 검출을 위한 패리티 비트나 문자 단위의 시작(Start) 비트, 정지(Stop) 비트를 붙여 불규칙하게 전송하는 방식은?
비동기식 전송 (Asynchronous Transmission)
Q15. 이더넷(Ethernet) 환경에서 쓰이는 매체 접근 제어 방식으로, 선로가 비어있는지 확인한 뒤 데이터를 보내고 충돌 여부를 감시하는 방식은?
CSMA/CD
Q16. 무선 LAN(Wi-Fi) 환경에서 쓰이는 매체 접근 제어 방식으로, 무선 특성상 충돌을 감지하기 어려워 사전에 충돌을 '회피'하는 방식을 취하는 것은?
CSMA/CA
Q17. 네트워크 구조(토폴로지) 중 중앙에 제어 노드가 있고 모든 컴퓨터가 1:1로 연결된 방사형 형태로, 하나의 선이 고장 나도 다른 컴퓨터에 영향이 없는 형태는?
성형 (Star 토폴로지)
Q18. 모든 노드가 그물망처럼 서로 다중 연결되어 있어, 선로 하나가 끊어져도 완벽한 우회로가 보장되는 가장 신뢰성 높은 네트워크 토폴로지는?
망형 (Mesh 토폴로지)
Q19. 전송 제어 헤더에 포함되는 에러 검출 기법 중 하나로, 전체 데이터 블록을 비트 단위로 다항식 연산하여 정밀하게 오류를 찾아내는 방식은?
CRC (Cyclic Redundancy Check / 순환 중복 검사)
Q20. 흐름 제어(Flow Control) 기법 중 하나로, 하나의 패킷을 보내고 수신 측으로부터 잘 받았다는 확인 응답(ACK)이 올 때까지 대기했다가 다음 패킷을 보내는 방식은?
Stop-and-Wait (정지-대기) 기법
Q21. 가상 회선을 설정하여 데이터의 흐름과 순서를 제어하고 고도의 신뢰성을 보장하는 연결형 4계층 프로토콜은?
TCP (Transmission Control Protocol)
Q22. 오버헤드가 적어 전송 속도가 매우 빠르며, 실시간 스트리밍 서비스 등에 주로 사용되는 비연결형 4계층 프로토콜은?
UDP (User Datagram Protocol)
Q23. 네트워크 계층 프로토콜로, IP 패킷 전송 중 발생하는 경로 에러 보고 및 네트워크 상태 진단(ping의 기반) 역할을 수행하는 것은?
ICMP (Internet Control Message Protocol)
Q24. 하나의 송신자가 네트워크 내의 '모든' 수신자에게 데이터를 동시에 전송하는 통신 방식을 지칭하는 용어는?
브로드캐스트 (Broadcast)
Q25. 특정 그룹에 속한 여러 수신자에게만 선택적으로 데이터를 전송하는 무선/화상 회의 통신 방식은?
멀티캐스트 (Multicast)
Q26. 멀티캐스트 그룹을 관리하기 위해, 호스트와 라우터 간에 그룹 가입/탈퇴 등의 정보를 주고받을 때 사용하는 프로토콜은?
IGMP (Internet Group Management Protocol)
Q27. 원격지 컴퓨터에 안전하게 접속하기 위한 터미널 에뮬레이터 프로토콜로, 암호화되지 않던 Telnet의 보안 취약점을 해결한 암호화 프로토콜은?
SSH (Secure Shell)
Q28. 인터넷 주소창의 영문 도메인 네임을 컴퓨터가 이해하는 숫자형 IP 주소로 변환해 주는 시스템은?
DNS (Domain Name System)
Q29. 네트워크 장비들을 중앙에서 모니터링하고 설정 정보를 수집 및 관리하기 위해 사용하는 응용 계층의 네트워크 관리 프로토콜은?
SNMP (Simple Network Management Protocol)
Q30. IP 주소를 동적으로 할당하고 회수하여, 사용자가 일일이 고정 IP를 세팅하지 않고도 인터넷을 바로 쓸 수 있게 해주는 자동 주소 할당 프로토콜은?
DHCP (Dynamic Host Configuration Protocol)
