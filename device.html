4과목 네트워크 운용기기 - 고빈출 20제 퀴즈

출제 문항 수는 적지만 난이도가 낮아 다 맞추고 들어가는 고효율 과목입니다.

📌 학습 가이드: 4과목(네트워크 운용기기)은 5문항이 출제됩니다. 리피터, 허브, 스위치, 라우터, 게이트웨이가 OSI 몇 계층 장비인지 식별하는 문제와 UTP 케이블 규격 및 광케이블 특성을 매칭하는 문제가 무조건 출제됩니다. 완벽하게 점수를 챙겨가세요!
Q1. 케이블을 길게 연결할 때 전송 신호가 약해지거나 왜곡되는 현상을 방지하기 위해, 약해진 전기 신호를 수신하여 재생 및 증폭한 후 다시 전달하는 OSI 1계층 장비는?
리피터 (Repeater)
Q2. 한 포트에서 수신한 전기 신호를 연결된 다른 모든 포트로 똑같이 카피해서 무조건 전송하는 특성(브로드캐스팅) 때문에 대역폭 낭비가 심한 1계층 집중 장치는?
더미 허브 (Dummy Hub)
Q3. 데이터 링크(2계층) 주소인 MAC 주소를 기반으로, 목적지 컴퓨터가 연결된 특정 포트로만 데이터 프레임을 정확히 필터링 및 전환하여 전송해 주는 똑똑한 집중 장치는?
스위치 (Switch) 또는 스위칭 허브
Q4. 네트워크 계층(3계층) 주소인 IP 주소를 식별하고, 서로 다른 네트워크 간에 데이터를 전송할 때 최적의 경로를 설정(Routing)해 주는 내비게이션 장비는?
라우터 (Router)
Q5. OSI 7계층 전 영역에 걸쳐 작동할 수 있으며, 서로 전혀 다른 프로토콜 체계를 사용하는 네트워크(예: LAN 세계와 메인프레임 세계)를 상호 연결해 주는 최외곽 '출입구' 장치는?
게이트웨이 (Gateway)
Q6. 스위칭 장비가 처리 장치(하드웨어) 방식이 아닌 소프트웨어 방식으로 데이터 링크 계층의 프레임을 전달하도록 설계된 고전적 2계층 연결 장비는?
브리지 (Bridge)
Q7. 라우터가 수행하는 기능 중 하나로, 들어온 패킷을 목적지 하위 네트워크로 안전하게 넘겨주기 위해 내부 경로 전환 테이블을 참조하는 동작을 무엇이라 하는가?
포워딩 (Forwarding)
Q8. 일반적인 2계층 스위치(L2 스위치)에 라우터의 기능인 3계층 IP 주소 판별 및 라우팅 기능까지 함께 탑재하여 고속 패킷 처리를 수행하는 장비는?
L3 스위치 (Layer 3 스위치)
Q9. 웹 트래픽(HTTP, HTTPS)이나 애플리케이션 프로토콜의 콘텐츠 내부 영역까지 깊숙이 분석하여, 사용자가 몰리는 서버들로 부하를 공평하게 분산(로드 밸런싱)해 주는 고성능 7계층 스위치는?
L7 스위치 (Layer 7 스위치)
Q10. 무선 랜(Wi-Fi) 환경에서 유선 네트워크 선로와 무선 주파수 신호를 중간에서 중계해 주어 노트북이나 스마트폰이 무선 인터넷에 접속할 수 있게 돕는 무선 기지국 장치는?
AP (Access Point / 무선 액세스 포인트)
Q11. 우리가 흔히 랜선이라고 부르는 케이블로, 두 가닥의 구리선을 꼬아서 외부 간섭을 줄이도록 설계되었으며 별도의 외부 피복 차폐막이 없는 저렴한 케이블의 약어는?
UTP 케이블 (Unshielded Twisted Pair)
Q12. UTP 케이블과 달리 구리선 주위를 금속 망이나 은박지와 같은 차폐막으로 겹겹이 감싸서, 공장이나 노이즈가 심한 가혹한 환경에 쓰이도록 절연 성능을 높인 케이블은?
STP 케이블 (Shielded Twisted Pair)
Q13. UTP 케이블을 컴퓨터나 스위치 포트에 꽂을 때 랜선 끝부분에 찍어서 연결하는 투명한 플라스틱 커넥터의 규격 명칭은?
RJ-45 커넥터
Q14. UTP 케이블 카테고리 규격 중, 기가비트 이더넷(1Gbps) 전송을 지원하여 현대 일반 가정이나 사무실에서 가장 대중적으로 쓰이는 규격 등급 명칭은?
카테고리 5e (Cat.5e) 또는 카테고리 6 (Cat.6)
Q15. 구리선 대신 유리를 원료로 한 가느다란 코어에 빛(광신호)을 쏘아 데이터를 전송하는 매체로, 대역폭이 매우 넓고 전자파 유도 노이즈의 영향을 전혀 받지 않는 초고속 전송 케이블은?
광케이블 (Optical Fiber Cable)
Q16. 광케이블 종류 중, 코어의 직경이 두꺼워 빛이 여러 경로로 반사되며 분산 전송되는 방식으로, 감쇄 현상 때문에 장거리 전송에는 불리하지만 가격이 저렴해 단거리 건물 내부에 주로 쓰이는 광케이블은?
멀티 모드 (Multi Mode) 광케이블
Q17. 광케이블 종류 중, 코어 직경이 매우 가늘어서 빛이 일직선(하나의 모드)으로만 곧게 통과하는 방식으로, 신호 감쇄가 적어 수십 km 이상의 장거리 광역 통신망(WAN)망 구축에 필수적으로 쓰이는 광케이블은?
싱글 모드 (Single Mode) 광케이블
Q18. 이더넷 규격 이름인 `100Base-TX`에서 맨 앞의 숫자 `100`이 의미하는 네트워크 전송 속도는 무엇인가?
100Mbps
Q19. 이더넷 규격 이름인 `10Base-T` 또는 `100Base-TX`에서 가운데 영문 단어 `Base`가 의미하는 신호 전송 방식 유형은?
베이스밴드 (Baseband / 디지털 신호를 그대로 직접 전송하는 방식)
Q20. 두 컴퓨터나 장비를 1:1로 직접 연결하기 위해, 한쪽 끝은 T568A 배열로 만들고 다른 쪽 끝은 T568B 배열로 꼬아서 배열을 엇갈리게 제작한 UTP 케이블의 종류 명칭은?
크로스 케이블 (Cross Cable)
